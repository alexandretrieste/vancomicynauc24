Vancomycin AUC24 Formulas and Explanations

Step 1: Calculate ke

ke = ln(Measured Peak / Measured Trough) / (T2 — T1)

Explanation: ke represents the elimination rate constant. It is calculated using the natural logarithm of the ratio of the measured peak to the measured trough concentration, divided by the time interval between the measurements.

Example: ke = 0.025 hr-1

Step 2: Calculate True Peak

True Peak = Measured Peak / e(—ke)(T1—Tinf)

Explanation: True Peak is the peak concentration adjusted for the start of infusion time. It is calculated using the measured peak concentration and the elimination rate constant.

Example: True Peak = 88.7 μg/mL

Step 3: Calculate True Trough

True Trough = (Measured Trough) * e(—ke)(Tau—T2)

Explanation: True Trough is the trough concentration adjusted for the end of the dosing interval. It is calculated using the measured trough concentration, the elimination rate constant, and the dosing interval.

Example: True Trough = 26.8 μg/mL

Step 4: Calculate AUCinf

AUCinf = (True Trough + True Peak) / 2 * (Tinf)

Explanation: AUCinf represents the area under the concentration-time curve from time zero to infinity. It is calculated as the average of the true trough and true peak concentrations multiplied by the infusion time.

Example: AUCinf = 57.8 μg/mL x hr

Step 5: Calculate AUCelim

AUCelim = (True Peak — True Trough) / ke

Explanation: AUCelim represents the area under the concentration-time curve due to elimination. It is calculated as the difference between the true peak and true trough concentrations divided by the elimination rate constant.

Example: AUCelim = 2431.6 μg/mL x hr

Step 6: Calculate AUC24

AUC24 = [(AUCinf) + (AUCelim)] * (24 / Tau)

Explanation: AUC24 represents the area under the concentration-time curve over the dosing interval. It is calculated as the sum of AUCinf and AUCelim, adjusted for the dosing interval.

Example: AUC24 = 1245 μg/mL x hr

Step 1: Calculate volume of distribution

VD = (Dose) * (1 — e—ke • Tinf) / (Tinf • ke(True Peak — [True Trough • e—ke • Tinf]))

Explanation: VD represents the volume of distribution. It is calculated using the dose, elimination rate constant, start of infusion time, true peak concentration, and true trough concentration.

Example: VD = 23.66 L

Step 2: Calculate vancomycin clearance (CL)

CL = VD • ke

Explanation: CL represents the clearance of vancomycin. It is calculated as the product of volume of distribution and elimination rate constant.

Example: CL = 0.60 L/hr

Step 3: Predict new peak concentration

New Peak = (New Dose / (CL • New Tinf)) * (1 — e—ke • New Tinf) / (1 — e—ke • New Tau)

Explanation: New Peak represents the predicted peak concentration after a new dose administration. It is calculated using the new dose, vancomycin clearance, new infusion time, and new dosing interval.

Example: Predicted Peak = 44.4 μg/mL

Step 4: Predict new trough concentration

New Trough = New Peak • e—ke(New Tau — New Tinf)

Explanation: New Trough represents the predicted trough concentration after a new dose administration. It is calculated using the predicted peak concentration, elimination rate constant, and new dosing interval.

Example: Predicted Trough = 13.4 μg/mL
